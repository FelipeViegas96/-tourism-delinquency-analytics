# 🌍 Tourism Delinquency Analytics

Projeto de análise financeira focado em pagamentos recorrentes e inadimplência 
em uma agência de turismo, desenvolvido com Power BI.

---

## 📊 Sobre o Projeto

Este projeto foi desenvolvido como parte do meu portfólio de Data Analytics, 
com o objetivo de demonstrar habilidades em visualização e análise de dados 
financeiros no setor de turismo.

---

## 📈 Visão Geral dos Dados

| Métrica | Valor |
|---|---|
| 💰 Receita Esperada (Parcelas) | R$ 38,4 Mi |
| 💵 Receita Real (Parcelas) | R$ 29,1 Mi |
| ⚠️ % Inadimplência Geral | 24,1% |
| 🛒 Total de Vendas | 169.591 |
| ✅ Vendas Pgto Normal | 119.457 |
| 🔄 Vendas Recorrentes | 50.134 |
| 📊 % Recorrente | 29,56% |
| 👥 Qtd Clientes | 130.821 |

---

## 🗂️ Páginas do Dashboard

- **Informações Gerais** — Visão consolidada de receita, inadimplência e vendas
- **Fluxo de Caixa** — Análise do fluxo financeiro da agência
- **Análise Cohort** — Comportamento de clientes por coorte
- **Análise Cohort Detalhada** — Detalhamento aprofundado por coorte
- **Análise Juros** — Impacto de juros na inadimplência
- **Tooltip Cohort** — Tooltips interativos para análise cohort

---

## 🔍 Principais Análises

- **% Inadimplência por Produto** — Pacote de Natal lidera com 34,92%,
  seguido de Passeio bate-volta (28,09%) e Fim de semana (26,34%)
- **Inadimplência por Estado** — Mapa de calor do Brasil e América do Sul
- **Inadimplência por Forma de Pagamento** — PIX (26,95%),
  Boleto (24,81%) e Cartão de Crédito (23,96%)
